### 📂 Portfólio de Decisão Multicritério (sad-mcdm)

Desenvolvo e mantenho a organização **[sad-mcdm](https://github.com/sad-mcdm)**. O projeto centraliza resolvedores matemáticos estruturados em diferentes linguagens e interfaces de usuário.

#### Resolvedores em Destaque:
*   [Projetos-de-MCDM](https://github.com/sad-mcdm/Projetos-de-MCDM): Repositório que une submódulos Git das ferramentas de cálculo.
*   [sad-mcdm-lib (Python)](https://github.com/sad-mcdm/sad-mcdm-lib) e [sad-mcdm-r (R)](https://github.com/sad-mcdm/sad-mcdm-r): Pacotes nativos com motores de cálculo para AHP, TOPSIS, BWM, MACBETH e PROMETHEE.
*   [sad-mcdm-js (JS/TS)](https://github.com/sad-mcdm/sad-mcdm-js): Implementação com resolvedor Simplex de duas fases rodando no navegador.
*   Bibliotecas em C++ (header-only com CMake), Julia (módulo SADMCDM) e Java (compatível com JDK 21).

#### Sistemas de Apoio à Decisão (SAD):
*   [sad-central-mcdm](https://github.com/sad-mcdm/sad-central-mcdm): Hub que processa matrizes de consequência e exporta relatórios executivos em PDF.
*   [sad-smarts-smarter](https://github.com/sad-mcdm/sad-smarts-smarter): Elicitação direta por Swings e centróides ordinais ROC.
*   [sad-bwm](https://github.com/sad-mcdm/sad-bwm) e [sad-bwt](https://github.com/sad-mcdm/sad-bwt): Solvers baseados em otimização minimax e taxas físicas de tradeoff.
*   [sad-promethee](https://github.com/sad-mcdm/sad-promethee): Resolvedores com suporte a restrições de mochila 0-1 (PROMETHEE V).
*   Módulos para os métodos AHP, MACBETH, TOPSIS, VIKOR e a família ELECTRE (I, IS, II, III, IV, TRI).
